# Omniglot (не дописан)

[Omniglot](https://www.omniglot.com/conscripts/natlangs.htm) – это онлайн-энциклопедия систем письма и языков, в том числе содержащая в себе отдельные наьоры фраз на разных языках и советы по изучению.

## ArchiveReady

![](/omniglot.com/ArchiveReady.png)

У [данного сайта](https://archiveready.com/check?url=https%3A%2F%2Fwww.omniglot.com%2Fconscripts%2Fnatlangs.htm) умеренно высокий рейтинг архивируемости. В отличие от других сайтов коллекции, у него крайне высокое соответствие стандартам (Standards Compliance), но при этом достаточно низкие доступность (Accessibility) и целостность (Cohesion), что критически сказывается на возможности его полноценного архивирования. Были обнаружены следующие проблемы с доступностью: наличие команд disallow в robots.txt, есть изображения, хранящиеся на других доменах. Помимо этого недоступны HTTP-заголовки, есть некорректные ссылки, HTML и CSS-файлы, а также Java-script вставки и скрипты с других доменов. Всё это делает полноценное архивирование сайта затруднительным.

## metawarc

`metawarc analyze lingvarium.org.warc.gz`

| mimes      | files | size   | share      |
|------------|-------|--------|------------|
| text/html  | 2     | 780190 | 99.9686    |
| text/plain | 1     | 245    | 0.0313927  |
| #total     | 3     | 780435 | 100        |
	
Закономерно, так как скачалась только основная страница-оглавление, бОльшую часть архива занимает её html-код (хотя и он не превышает даже мегабайта).
	 
`metawarc metadata --output meta.jsonl lingvarium.org.warc.gz`

Никаких метаданных сохранить не получилось по той же причине.
 
`metawarc index  lingvarium.org.warc.gz`
 
Эта команда индексирует архив и создаёт базу данных, по которой можно производить дополнительный анализ.

`metawarc stats -m mimes`

| mime                     |   size | count |
|--------------------------|-------:|------:|
| text/html                |  10564 |     1 |
| text/html; charset=utf-8 | 769626 |     1 |
| text/plain               |    245 |     1 |

Таблица практически дублирует информацию команды analyze выше.
