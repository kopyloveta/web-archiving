# Lingvarium project

[Lingvarium](http://lingvarium.org/index.shtml) – это сайт, созданный лингвистом Юрием Коряковым. Он содержит самую разнообразную информацию о языках мира: карты, реестр языков, сведения о письменностях и фонологии различных языков.

## ArchiveReady

![](/lingvarium.org/ArchiveReady.png)

У [данного сайта](https://archiveready.com/check?url=http%3A%2F%2Flingvarium.org%2Findex.shtml) средний рейтинг архивируемости: наиболее низкий показатель у параметра доступности (Accessibility), у остальных параметров показатели также не очень высокие. У сайта отсутствует sitemap, в файле robots.txt есть команды disallow, закрывающие доступ к некоторым областям сайта. Существуют изображения, прикрепленные с других сайтов, есть устаревшие html-теги, а также java-script вставки, также осложняющие полноценную архивацию сайта.

## Replay Webpage

## metawarc

`metawarc analyze lingvarium.org.warc.gz`

| mimes      | files | size   | share      |
|------------|-------|--------|------------|
| text/html  | 2     | 780190 | 99.9686    |
| text/plain | 1     | 245    | 0.0313927  |
| #total     | 3     | 780435 | 100        |
	
Закономерно, так как скачалась только основная страница-оглавление, бОльшую часть архива занимает её html-код (хотя и он не превышает даже мегабайта).
	 
`metawarc metadata --output meta.jsonl lingvarium.org.warc.gz`

Никаких метаданных сохранить не получилось по той же причине.
 
`metawarc index  lingvarium.org.warc.gz`
 
Эта команда индексирует архив и создаёт базу данных, по которой можно производить дополнительный анализ.

`metawarc stats -m mimes`

| mime                     |   size | count |
|--------------------------|-------:|------:|
| text/html                |  10564 |     1 |
| text/html; charset=utf-8 | 769626 |     1 |
| text/plain               |    245 |     1 |

Таблица практически дублирует информацию команды analyze выше.
